🔴 Problèmes rencontrés et solutions:

        1. Détection des bots (ex: Cloudflare)

                Problème :
                    Le site bloque le scraper (CAPTCHA, accès refusé).

                Solution :

                    Utiliser des User-Agents différents
                    Ajouter des délais entre les requêtes
                    Utiliser un navigateur automatisé (ex: Playwright)
            

        2. Changement de structure du site

                Problème :
                    Le code ne fonctionne plus après modification du HTML.

                Solution :

                    Utiliser des sélecteurs flexibles
                    Ajouter try/except
                    Surveiller les changements du site

        3. Normalisation de la base de données

                Problème :
                    Données dupliquées et désorganisées.

                Solution :

                    Appliquer la normalisation (jusqu’à 3NF)
                    Séparer les tables (produits, prix, magasins)
                    Utiliser des clés primaires et étrangères

        4. Local vs Cloud

                Problème :

                    Local : le PC doit rester allumé
                    Cloud : plus complexe

                Solution :

                    Développement en local
                    Automatisation avec Task Scheduler
                    Prévoir un déploiement cloud plus tard
